# Reddi Manoj - Modern Portfolio Website

A stunning, modern portfolio website built with HTML5, CSS3, and JavaScript featuring glassmorphism design, smooth animations, and responsive layout.

## 🚀 Features

- **Modern Glassmorphism Design**: Beautiful frosted glass effects with backdrop blur
- **Responsive Layout**: Optimized for all devices (desktop, tablet, mobile)
- **Smooth Animations**: CSS transitions, particle effects, and scroll animations
- **Interactive Elements**: Typing animation, hover effects, and dynamic backgrounds
- **Dark Theme**: Cyberpunk-inspired color scheme with neon accents
- **Performance Optimized**: Fast loading with optimized assets

## 🎨 Design Highlights

- **Color Palette**: Dark theme with cyan (#00f0ff), purple (#8a2be2), and pink (#ff007f) accents
- **Typography**: Inter font family for modern, clean text
- **Animations**: Floating particles, glowing lines, and smooth transitions
- **Glassmorphism**: Backdrop blur effects with subtle borders

## 📱 Sections

1. **Hero**: Eye-catching introduction with profile photo and typing animation
2. **About**: Personal information with animated statistics
3. **Skills**: Technical skills organized by category
4. **Projects**: Showcase of key projects with links and technologies
5. **Training**: Data science and analytics training details
6. **Certificates**: Professional certifications and achievements
7. **Contact**: Contact form with validation and social links

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Modern styling with Grid, Flexbox, and animations
- **JavaScript**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Inter font family

## 🚀 Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. For local development, run a local server:
   ```bash
   python3 -m http.server 8080
   ```
4. Open `http://localhost:8080` in your browser
